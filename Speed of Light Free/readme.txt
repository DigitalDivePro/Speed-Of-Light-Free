== Description ==

Speed of Light (Free) is an edge-first Cloudflare caching layer that keeps WordPress behavior intact. It deploys safe cache rules and headers so eligible pages are served from the Cloudflare edge without risky optimizations.

Built by DigitalDive (https://digitaldive.pro).

Highlights:

- Cloudflare edge caching with safe defaults for logged-in users, POST requests, and common ecommerce flows.
- Simple controls for cache rules, TTL, and exclusions.
- Manual purge tools for entire cache, homepage, or specific URLs.

Looking for local HTML caching and Turbo Preload? Speed of Light Pro is available at https://speedoflight.pro.

== Installation ==

1. Upload the `speed-of-light-free` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Go to Speed of Light Free and enter your Cloudflare credentials.

== Frequently Asked Questions ==

= Do I need a Cloudflare account? =
Yes. This plugin manages Cloudflare cache rules for your zone.

= Is there a Pro version? =
Yes. Speed of Light Pro adds local HTML caching and Turbo Preload. Learn more at https://speedoflight.pro.
