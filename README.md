# Speed of Light (Free)

Conservative Cloudflare full-page caching for WordPress that is safe for any site.

Features
- Cloudflare edge-first caching with safe defaults for logged-in users and ecommerce flows
- Simple controls for cache rules, TTL, and exclusions
- Manual purge for full site, homepage, or specific URLs

Installation
1. Upload the `speed-of-light-free` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Visit the plugin settings and enter your Cloudflare credentials / zone details.

Compatibility
- Requires WordPress 5.8 or later
- Tested up to WordPress 6.9
- Requires PHP 7.0 or later
